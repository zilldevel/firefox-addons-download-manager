## Installation:

On Fedora Linux:

    sudo dnf install comic-neue-fonts
    sudo dnf install gnu-free-sans-fonts
    sudo dnf install liberation-sans-fonts
    sudo dnf install mozilla-fira-mono-fonts
    sudo dnf install mozilla-zilla-slab-fonts
